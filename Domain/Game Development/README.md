# GAME DEVELOPMENT

**OBJECTIVE: BEGINNER ENVIRONMENT DESIGN**

For this task, you need to create a simple and creative environment using Unity. You can import free assets from the Unity Asset Store and design any type of scene you like such as a park, village, fantasy world, city, island, sci-fi area, etc.

This task is only about **environmental design**. No coding, gameplay, or character movement is required. Focus on **creativity, proper placement of assets, lighting, and overall look of the scene**.

---

**SUBMISSION REQUIREMENTS:**

1. Unity project file or exported package

2. 3–5 screenshots

3. A short description explaining your concepts
