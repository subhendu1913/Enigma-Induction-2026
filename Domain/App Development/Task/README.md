# APP DEVELOPMENT

**PROJECT OVERVIEW:**

Build a simple **Notes Taking Android App** similar to Google Notes.  
The aim is to learn **basic Android development** by creating a practical app.

---

**Your app must include (Mandatory Features):**

1. Create notes with checklist (to-do style)  
2. Add voice recording to notes  
3. Attach images to notes  
4. Simple and minimal Material UI  
5. Custom app icon  
6. App opening animation  
7. Animation when opening a note  
8. Add, edit, and delete notes  

---

**Implement at least one extra feature of your choice:**

1. Dark mode  
2. Camera image capture  
3. Export notes as PDF  
4. Home screen widget etc.  

---

**SUBMISSION GUIDELINES:**

You must:

● Document your progress  
● Push code regularly to GitHub  
● Use proper commit messages  

---

**TIPS:**

No Tech Stacks restriction.
