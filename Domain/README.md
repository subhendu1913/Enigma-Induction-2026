# ENIGMA INDUCTION TASK 2026

## COMPULSORY FOR ALL:

1. **Learn Git and GitHub Concepts** – Understand how Git tracks changes and how GitHub hosts and manages repositories online.

2. **Version Control** – Learn how version control helps in tracking changes, managing history, and collaborating on projects.

3. **Basic Git Commands (git init, git add, git commit, etc.)** – Practice essential commands used to create repositories, track files, and save project changes.

4. **Learn About Branches** – Understand how branches allow parallel development and safe experimentation.

5. **Forking and Cloning** – Learn how to copy repositories and contribute to projects using GitHub workflow.

6. **Hosting Sites on GitHub Pages** – Deploy and manage a simple website using GitHub Pages.
